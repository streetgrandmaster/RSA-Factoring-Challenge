# RSA-Factoring-Challenge

RSA Factoring Challenge

## Tasks 🚨 🚨 🚨

0.  __Factorize all the things!:__ Factorize as many numbers as possible into a product of two smaller numbers.

1. __RSA Factoring Challenge:__ RSA Laboratories states that: for each RSA number n, there exist prime numbers p and q such that n = p × q. The problem is to find these two primes, given only n.
